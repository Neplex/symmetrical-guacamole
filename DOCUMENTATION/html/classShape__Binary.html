Detailed Description

template<bool(*)(bool const ,bool const ) op>
class Shape_Binary< op >

this template provides a generic ways to generate operators between shapes such that:

  • the operator is binary, i.e. depends only on two shapes (an no other parameter)
  • the return value of contains only depends on the values returned by the contains (on the same point) in the two shapes.
